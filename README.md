# SMART PILL BOX USING BLUEPILL ARM COTEX M3
Independently designed and developed a smart pill box using the STM32 (Blue Pill) microcontroller to automate medication management. 

Integrated sensors for precise tracking, and implemented I2C and UART communication protocols for seamless interaction between components. Enabled Bluetooth connectivity to send real-time notifications via a mobile app.
